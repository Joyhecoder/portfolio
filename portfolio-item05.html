React project Exercise Buddy

Built by Manjuan

This is a frontend project built with JavaScript, React, Redux, Restful API, Google API, HTML, CSS, Bootstrap.

In this app, users can search different types of exercises and add their favorite ones into their lists. Each exercise comes with detailed information including trained muscle part, difficulty levels, required equipment, instruction and a video for user to follow along.
