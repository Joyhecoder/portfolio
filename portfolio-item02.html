Backend project Blackpink band website

Built by Manjuan

This is a backend project built with JavaScript, Node, EJS, Express, Socket.io, HTML, CSS, Bootstrap.

The website allows users to look through all the albums from Blackpink and on each album's details page, users can play the tracklists on the website. Users can also leave feedback on the web page and the feedback is stored in a Json file. Last but not least, users can start online live chat and message other users who are also on the website. The chat feature was built with socket.io.
